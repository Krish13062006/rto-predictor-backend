Quick setup and run instructions

1) Install Node.js (LTS)

- Windows (winget):

  winget install OpenJS.NodeJS.LTS

- Or download from: https://nodejs.org/

2) From project folder:

  npm install
  npm start

Notes:
- `package.json` already includes dependencies `express`, `pg`, and `dotenv`.
- If you prefer `nvm`, use that to install Node and switch to an LTS version.
